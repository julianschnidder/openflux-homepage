OpenFLUX

About the Software

OpenFLUX is a MATLAB-based modelling software for 13C metabolic flux analysis (MFA). Modern techniques like EMUs are implemented for efficient computing. A typical MFA with OpenFLUX consists of model set up (in a spreadsheed application like OpenOfficee.org Calc or MS Office Excel), a model generation using the Java GUI distributed with OpenFLUX, and the analysis itself in Matlab. A series of screenshots depicts this procedure.

OpenFLUX is released under the GNU GPL licensing condition. However, we would also like to keep track of users for the purpose of gathering feedback (for further improvement of the software). Please kindly download and fill in the registration file (see "Additional files"), and then email it back to us.

Requirements and Installation

For running this software you need an installation of Java™ and a recent version of Matlab® including the Optimization and Statistics Toolbox™.

To download, please direct your browser to the download section. Please also tell us about your use of OpenFLUX.

The Java part that builds the model from the spreadsheet is started by double clicking the jar-file.

In Matlab you add the path to the directory with the m-files to the Matlab path (e.g. with pathtool). The MFA is started by start13OF.

For more information see the user manual.
